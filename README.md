## 🎬 Netflix Movie Catalog Analysis (2012–2016)

### 📌 Overview

This project analyzes movie data from 2012–2016 to help **Netflix** determine the most profitable genres when expanding its catalog of older movies.

Using **Tableau and data analysis**, we explore how **budget, revenue, and profitability vary across genres** to support data-driven investment decisions.

---

### 🎯 Objective

* Identify **high-ROI genres**
* Compare **budget vs revenue trends**
* Recommend **cost-effective content strategies for Netflix**

---

### 🛠️ Tools & Technologies

* Tableau (Data Visualization)
* Excel / Google Sheets (Data Cleaning)
* Python (optional, if you include notebook)
* GitHub (Project Documentation)

---

### 📊 Key Insights

* **Action, Adventure, and Comedy** generate the highest total revenue
* **Horror and Documentary** films offer **high ROI with low budgets**
* **Crime and Biography** genres show lower profitability
* High-budget films drive revenue, while low-budget films maximize efficiency

---


### 📂 Dataset

* Source: Movie dataset (2012–2016)
* Includes: Genre, Budget, Revenue, Cast, Directors

---

### 💡 Recommendations

* Invest in **Action & Adventure** for large-scale revenue
* Include **Horror & Documentary** for cost-efficient returns
* Balance portfolio between **high-budget and low-budget genres**
